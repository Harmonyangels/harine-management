Strategic Note on Color
Healthcare analytics is dominated by navy and teal. Harine's crimson-terracotta palette is a deliberate differentiator — it signals confidence and conviction rather than blending in with the sea of blue dashboards. For executive-facing materials (proposals, presentations), use crimson as the dominant color. For dashboards and data-heavy interfaces, use the ink neutrals as the dominant with crimson as an accent.

04 — Voice & Tone
The operator who
speaks data.

Harine's voice comes from a position of lived experience — not theory. We've run healthcare businesses. We know what matters to a CMO on a Monday morning and what a PE associate needs in a data room.

🎯
Direct
We say what we mean. No jargon for jargon's sake. If a sentence can be shorter, it should be. Executives respect brevity.
🏗️
Grounded
Our credibility comes from having built real things. We reference real problems — manual exports, reconciling reports, provider comparisons in spreadsheets.
⚡
Confident
We know the answer. No hedging. "We help" not "we try to help." "You'll see" not "you might see." Own the outcome.
🚫
Not Salesy
We don't oversell. The work speaks. Testimonials and case study outcomes do more than adjectives like "world-class" or "cutting-edge."
🚫
Not Academic
We have the engineering background but we don't lead with it. Plain language first. Technical depth is available on request.
🚫
Not Buzzword-Heavy
Words like "synergy," "leverage," "holistic," "empower" are banned. Describe the actual thing, not a vague version of it.

Voice in Practice

✕ Don't write this
"We leverage cutting-edge AI-powered solutions to transform your healthcare data ecosystem into actionable insights that empower leadership."
✓ Write this instead
"We take your EHR data and build dashboards your leadership actually opens every morning."
✕ Don't write this
"Our comprehensive due diligence analytics offering provides holistic visibility into target practice performance metrics."
✓ Write this instead
"Before you close, you need to know if the revenue is real. We find out."
✕ Don't write this
"Harine Management is a premier provider of world-class healthcare data analytics consulting services."
✓ Write this instead
"We've run healthcare businesses. We built the data systems we wished we had. Now we build them for you."

A Note on Logo vs. Website Color
Your logo uses crimson and terracotta. The initial website mockup used forest green — a deliberate "executive trust" choice. Both work. The recommended approach: use crimson/terracotta as the brand identity color (logo, proposals, slides, email headers) and allow the website to use the warmer neutral palette (stone, cream, ink) with crimson accents. This prevents the site from feeling like "just a red website" while keeping the logo as the bold, memorable identity anchor it is.
